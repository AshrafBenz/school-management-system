# 🎓 School Management System

## 📌 Description
This is a web application developed to manage a school system.  
It allows administrators to manage students, classes, and authentication.

## ⚙️ Technologies Used
- PHP
- MySQL
- HTML
- CSS

## 🚀 Features
- 🔐 Authentication system (Login / Register / Logout)
- 👨‍🎓 Manage students (Add / Edit / Delete)
- 🏫 Manage classes
- 📊 Organized admin panel

## 📁 Project Structure
- Admin/ → Admin management
- authentication/ → Login & register system
- css/ → Styles
- includes/ → Header & footer
- eleves/ → Student management

## 🛠️ Installation
1. Clone the repository
2. Import the database file (.sql) into phpMyAdmin
3. Configure database connection in `connexion.php`
4. Run the project on localhost (XAMPP)
